b) Explique o que uma string e como ela é tratada em C.

    Uma string é uma sequência de caracteres em um vetor, podendo ser letras ou outros como números e caracteres especiais. Pra se utilizar de uma string, usamos a variável tipo CHAR e ela sempre vem com uma ocupação de bit final 0, então se uma string tiver 4 letras, ocupará os espaços 0, 1, 2, 3 e 4. Cinco espaços, pois o último sempre será vazio.
